# Bank Marketing Classification Project

## Overview
This project uses the UCI Bank Marketing dataset to predict whether a client will subscribe to a term deposit. Six machine learning models are implemented and evaluated.

## Models Implemented
1. Logistic Regression
2. Decision Tree
3. K-Nearest Neighbors
4. Naive Bayes
5. Random Forest
6. XGBoost

## Evaluation Metrics
- Accuracy
- AUC Score
- Precision
- Recall
- F1 Score
- Matthews Correlation Coefficient (MCC)
